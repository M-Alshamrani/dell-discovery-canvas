## Bugs caught + fixed during smoke (per discipline locked in feedback_*)

Two regressions were caught at smoke time and fixed before commit:

1. **Commit 25 → 26 transition**: initial sweep treated unloaded-catalog as "every catalog FK orphaned" → quarantined every record with a layerId or envCatalogId reference. 6 V-INT/V-DRIFT tests failed at smoke ("Cannot read properties of undefined"). Fix: `resolveTargetSet` returns `null` (skip check) for unloaded catalogs instead of empty `Set` (treat as empty target). Memory: `feedback_dockerfile_whitelist.md` + the broader DOM-first lesson.

2. **Commit 18 → 19 transition (V-PERF setup)**: reference engagement initially used `engagementId: "00000000-0000-4000-8000-acme00000000"` — `m` is a non-hex character. 7 V-PERF tests failed UUID validation. Fix: `"...ace000000000"`. Same DOM-first discipline.

3. **Commit 26 → 27 transition (LINKED chip label)**: `LINKED_BY_KIND` constant in `manifestGenerator.js` was missing `source: "linked"` on each entry → chips rendered with "UNDEFINED" label instead of "LINKED". Fix: added `source: "linked"` to all 7 linked-path entries. Caught at first browser-smoke screenshot.
